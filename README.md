# next-boilerplate

## Stack

* **Next.js**
* **Typescript**
* **Yarn**
* **styled-components**
* **next-i18next**
* **next-auth**
* **next-pwa**
* **ESLint**
* **Prettier**
* **Husky**
* **lint-staged**

## Setup

```bash
# Setup environment variables
mv .env.local.example .env.local

# Start development server
yarn dev
```
